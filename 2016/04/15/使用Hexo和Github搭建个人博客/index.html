安装Hexo

cd到要建博客的本地目录(如:H:\Hexo\blog)

然后又是刷屏等待……

OK，安装成功，赶紧看看hexo博客的效果先

打开浏览器，输入:http://localhost:4000 如果显示出了正常的博客，就说明在本地大家成功了。

注册Github账号

Github官网
点击右上角sign up（注册），如果已经有账号了，那就sign in。注册过程很简单，用户名、邮箱、密码即可。
然后前往自己刚才填写的邮箱，点开Github发送给你的注册确认信，确认注册，结束注册流程。一定要确认注册，否则无法使用gh-pages！

登陆之后，点击页面右上角的加号，选择New repository
填写信息

  • project name：2xxy.github.io
  • description： personal blog
  • 注：Github Pages的Repository名字是特定的，2xxy，那么我Github Pages Repository名字就是2xxy.github.io。

添加SSH Key到Github

  1. 首先你需要拷贝id_rsa.pub文件的内容，你可以用编辑器打开文件复制(如果看不到这个文件，你需要设置显示隐藏文件)，也可以用git命令复制该文件的内容，如：

  2. 登陆github系统。点击右上角的 Account Settings—>SSH And GPG keys —> New SSH Key

  3. 把你复制的 SSH key 代码粘贴到 key 所对应的输入框中，记得 SSH key 代码的前后不要留有空格或者回车。当然，上面的Title所对应的输入框你也可以输入一个该SSH key显示在github上的一个别名。默认的会使用你的邮件名称。
